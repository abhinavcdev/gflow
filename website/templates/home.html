WORKFLOW

Three commands.
One workflow.

1

Start a branch

Create a feature, bugfix, or hotfix branch with the right naming convention.

gflow start feature auth
2

Create a PR

Commit, push, and open a pull request in one command.

gflow pr "Add user auth"
3

Merge & clean up

Merge the PR, switch back to main, and delete the feature branch.

gflow finish
4

Monitor status

Check your branch status, CI checks, and sync state anytime.

gflow dash
